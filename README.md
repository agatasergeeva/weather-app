# Приложение для прогноза погоды

## Задача
Необходимо создать веб-приложение с прогнозом погоды, в котором пользователь сможет:
* При первом открытии приложения, будет запрошен доступ к геопозиции пользователя, и будет выведена прогноз на сегодня, а так же минимум на 2 следующих дня (В интерфейсе не нужно указывать что это за локация, можно написать "Текущее местоположенее")
* При отклонении разрешения доступа к гео-данным, появится форма добавления города для получения прогноза погоды (с идентичными требованиями)
* У пользователя так же должна быть возможность добавить как минимум 2 других города для получения информации о погоде в них
* В интерфейсе должна быть предусмотрена кнопка "Обновить" по которой будет заново запрашиваться информация о погоде (искуственное обновление страницы не считается, нужно заново отправить запросы)
* При перезагрузке приложения, информация о текущем местоположении (или вручную введеном городе) должна остаться, как и дополнительные города которые ввел пользователь

## Технические требования
* Запрещается использовать CSS-фреймворки (Bootstrap, Foundation и т.д.) и JS-фреймворков (Angular, React, Vue и т.д.).
* Разрешается использование препроцессоров CSS (SCSS) по желанию
